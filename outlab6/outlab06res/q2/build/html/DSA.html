class DSA.Heap(cap)[source]¶

Bases: object

This is the implementation of the data structure of a heap. A heap data structure has a structural property and a heap property. The structural property states that only the last level of the heap may be incomplete and all the nodes must be flushed towards the left i.e. the new nodes must always be added in the left most free space. The heap property states that the parent must be smaller than both its children.
